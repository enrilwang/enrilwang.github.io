# enrilwang.github.io

这是一个轻量级的 GitHub Pages 站点骨架（示例）。

本仓库当前包含一个简单的静态站点：

- `index.html` — 主页模板
- `assets/css/styles.css` — 站点样式

本地预览：

```bash
# 在仓库根目录运行一个简单的静态服务器，然后在浏览器打开 http://localhost:8000
python3 -m http.server 8000
```

部署建议：

- 直接使用 GitHub Pages（仓库的 `main` 分支或 `gh-pages` 分支，根据仓库设置）。
- 如果你想使用构建流程（例如 Jekyll / Hugo / Vite），请先在 Issue 中确认引入新工具的许可与回退方案。
